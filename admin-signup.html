대시보드에서 직접 만들기

  1. Supabase 프로젝트 → Authentication → Users
  2. Add user → Create new user
  3. 이메일·비밀번호 입력 후 생성(필요 시 “Auto Confirm User” 체크)
  4. admin.html에서 동일 계정으로 로그인
